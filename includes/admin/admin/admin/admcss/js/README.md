# Portal Universitario

Sistema web para gestión de contenido académico para estudiantes universitarios.

## Características

- **Dos tipos de usuarios**: Estudiantes y Administradores
- **Gestión de materias y documentos**
- **Panel de administración completo**
- **Interfaz responsive y moderna**

## Instalación

1. Clona el repositorio en tu servidor web
2. Crea una base de datos MySQL y ejecuta el archivo `database.sql`
3. Configura las credenciales de la base de datos en `includes/config.php`
4. Asegúrate de que el directorio `uploads/` tenga permisos de escritura
